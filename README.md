# Jumia Smart TVs Analytics Dashboard 📺

An interactive web application built with Python and Streamlit to analyze price distributions, brands market share, and discount strategies for Smart TVs on Jumia Egypt.

## Features
- **Dynamic Filtering:** Select any brand to view its specific data.
- **Live KPIs:** Real-time calculation of average prices and maximum discount values.
- **Data Visualization:** Interactive scatter plots demonstrating the relationship between current prices and discount amounts.
